## YouTube Video Downloader

This Python script provides a simple command-line interface to download YouTube videos either as MP3 audio files or MP4 video files. It utilizes the `pytube` library to fetch and download videos from YouTube.

### Features:

- **Flexible Download Options**: Choose between downloading videos as MP3 audio or MP4 video, based on user input.
- **Video Quality Selection**: When downloading videos as MP4, users can select from available video qualities.

### Installation:

1. Clone this repository by command `git clone https://github.com/PoplajZ/Youtube-videos-downloader`
2. Install the required dependencies by running command: `pip install pytube`.

### Usage:

1. Run the script by command `python3 YouTube_Videos_Downloader.py`
2. Input the URL of the YouTube video you want to download.
3. Choose whether to download the video as MP3 or MP4.
(4. If MP4 is chosen, select the desired video quality.)
5. The script downloads the video or audio file to the Downloads directory.

### Note:

This script is intended for personal use only and should be used in compliance with YouTube's terms of service.
